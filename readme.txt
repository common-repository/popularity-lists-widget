== Description ==

Popularity Lists Widget is a wordPress widget, Operates by using [Popularity Contest](http://wordpress.org/extend/plugins/popularity-contest/) Plugin. And, a popular article is output as a list.


== Installation ==

= Attention =
Please install [Popularity Contest](http://wordpress.org/extend/plugins/popularity-contest/) before installing this plugin. 

1. Unziped and Upload directory popularity-lists-widget to the /wp-content/plugins/ directory
1. Go to WP admin -> plugins and activate Popularity Lists Widget.
1. Go to WP admin -> presentation -> sidebar widgets, to add the widget to your sidebar(s).
1. Select how many Popularity Lists Widget you want to show and/or setup the parameters of the widget.
